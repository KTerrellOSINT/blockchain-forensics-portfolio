## Repository Contents

### Python Tools
- **redflag_scorer_v3.py** — A seven-indicator Bitcoin wallet red-flag scoring framework. Indicators include drain-to-zero detection, CV-based deposit clustering analysis, value-weighted sweep dominance, dormancy gap measurement, consolidation/fan-out pattern recognition, repeat counterparty dampening, and known mixing-service contact detection. Calibrated through testing against four real-world cases.

### Case Studies
